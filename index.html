📖 How to Use This Media Library

Uploading files: Click the upload area or drag and drop your images (JPG, PNG, GIF, WebP) or MP4 videos. Each file must be under 92MB.

Managing files: Each uploaded file appears as a card. You can:

  • Copy a single URL: Click on the thumbnail or the file name – the direct link is copied to your clipboard.
  • Copy all URLs on the current page: Use the “Copy All Links” button in the toolbar. All URLs from the current page (up to 50 items) are copied, one per line.
  • Preview: Click the file name to open a larger view with playback controls for videos.
  • Delete: Remove a file from your local list (the cloud copy remains intact).

Pagination: Use the Previous/Next buttons to navigate through your collection. The library stores your uploaded files in your browser’s local storage.

Free & secure: All media is hosted on NNEPIE with automatic optimization (q_auto,f_auto). Your files are private to your device – we do not store them on our servers.

❓ Frequently Asked Questions

What file formats are supported?
We support JPG, PNG, GIF, WebP for images and MP4 for videos. Other formats are not accepted.

Is there a file size limit?
Yes, the free Nnepie plan limits each file to 92 MB. Larger files will be rejected.

How long are my files stored?
Files are stored on Nnepie with your upload preset. They remain available as long as NNEPIE keeps them. This library does not delete your cloud files; it only manages the list in your browser’s local storage.

Can I share my files with others?
Absolutely – each file gets a permanent, shareable URL. Use the copy functions to get the link and share it anywhere.

What happens when I clear the list?
The “Clear All” button removes all entries from your local list, but does not delete the actual files from Nnepie. You can re-add them later by uploading again or by manually re-adding the URLs (not yet supported).

Why is there a proxy domain (img.nnepie.com)?
We use a proxy to add automatic image optimization (q_auto,f_auto) for faster loading and better performance. The original Nnepie URLs are still accessible.

Is this service free?
Yes, this media library is completely free to use for personal and commercial purposes, powered by Nnepie’s free tier.
